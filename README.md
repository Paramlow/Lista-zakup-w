# Lista zakupów – jQuery DOM Manipulation

## Funkcje aplikacji
- Dodawanie produktów (na początku, na końcu)
- Usuwanie ostatniego elementu
- Wyczyść i przywróć domyślną listę
- Edycja elementów listy (kliknięcie → input → Enter)
- Efekty wizualne (fadeOut / fadeIn)
- Sortowanie alfabetyczne A–Z
- Filtrowanie treści
- Kolorowanie co drugiego elementu
- Drag & Drop przy pomocy jQuery UI

## Użyte metody jQuery
- append(), prepend(), html(), text()
- remove(), empty()
- css(), toggle(), fadeIn(), fadeOut()
- sortable() z jQuery UI
- :even selektor
- Manipulacja eventami: click(), keypress()
